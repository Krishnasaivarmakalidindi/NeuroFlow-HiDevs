# NeuroFlow

NeuroFlow is a production-grade Retrieval-Augmented Generation (RAG) and AI workflow platform designed for document ingestion, retrieval, generation, evaluation, and fine-tuning.

## Features

- Multi-modal document ingestion
- Hybrid retrieval pipeline
- LLM routing
- Evaluation framework
- Fine-tuning pipeline
- Experiment tracking
- Model optimization

## Architecture

The project consists of five core subsystems:

- Ingestion
- Retrieval
- Generation
- Evaluation
- Fine-Tuning

Documentation is available in the `/docs` folder.
